Covid-19 Sentiment Analysis

Welcome to our Sentiment Analysis of tweets related to Covid-19! We were able to create several graphics that represent out findings. If you are interested in more information, please refer to our Github repository.

Link to Github Repository

Our hypothesis is that sentiment toward the Covid-19 vaccination program has been improving over the months since the announcement of the vaccine and FDA Emergency Use Authorization approvals of the first three U.S. vaccines; as well as the use of numerous other vaccines in nations worldwide. By analyzing tweets by keyword and utilizing sentiment analysis, we will deploy an app that will allow users to choose a date, or range of dates, to gain an understanding of the changing nature of Twitter users' attitudes toward Covid-19 vaccines.

We were able to create this project based on data from kaggle. The Twitter API is limited in calls and expensive to gain full access. We utilized the dateset based on the link below.
